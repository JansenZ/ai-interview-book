## 📐 1. 线性代数（Vector & Matrix）

### 向量 (Vector)

**直观理解**：一组数字，表示一个事物的特征

```javascript
// 前端类比：一个对象的所有属性
const user = { age: 25, height: 175, weight: 70 };
// 向量表示：[25, 175, 70]

// AI 应用：文本向量化（Embedding）
"你好" → [0.1, -0.5, 0.3, ..., 0.2] // 768 维向量
```

**面试考点**：
> **Q: 什么是 Embedding？**
> 
> **A**: 把文本（或图片）转换成一个向量（一串数字）。
> 语义相似的文本，向量也相似（距离近）。
> 用于 RAG 检索、语义搜索等场景。

### 矩阵 (Matrix)

**直观理解**：向量的数组，批量处理数据

```javascript
// 前端类比：二维数组
const users = [
  [25, 175, 70],  // 用户 1
  [30, 180, 75],  // 用户 2
  [28, 165, 60],  // 用户 3
];

// AI 应用：神经网络就是矩阵运算
输出 = 激活函数 (输入 × 权重 + 偏置)
```

### 向量运算

| 运算 | 公式 | 直观理解 | 前端应用 |
|------|------|----------|----------|
| **点积** | a·b = Σ(aᵢ × bᵢ) | 两个向量的相似度 | 计算文本相似度 |
| **余弦相似度** | cosθ = (a·b) / (|a||b|) | 夹角余弦，-1 到 1 | RAG 检索 |
| **欧氏距离** | √(Σ(aᵢ-bᵢ)²) | 空间中的直线距离 | 聚类分析 |

**代码示例**：
```javascript
// 计算余弦相似度（RAG 检索用）
function cosineSimilarity(a, b) {
  const dotProduct = a.reduce((sum, _, i) => sum + a[i] * b[i], 0);
  const normA = Math.sqrt(a.reduce((sum, val) => sum + val * val, 0));
  const normB = Math.sqrt(b.reduce((sum, val) => sum + val * val, 0));
  return dotProduct / (normA * normB);
}

// 结果越接近 1，表示越相似
cosineSimilarity([0.1, 0.2], [0.1, 0.2]) // → 1.0 (完全相同)
cosineSimilarity([0.1, 0.2], [0.9, 0.8]) // → 接近 0 (不相似)
```

**面试考点**：
> **Q: RAG 检索是怎么找到相关文档的？**
> 
> **A**: 
> 1. 把用户问题转换成向量（Embedding）
> 2. 计算问题和所有文档向量的余弦相似度
> 3. 选相似度最高的几个文档
> 4. 拼接到 prompt 里让 AI 回答

## 📈 2. 微积分（Calculus）

### 导数 (Derivative)

**直观理解**：变化率，函数在某点的斜率

```
f(x) = x²
f'(x) = 2x  // 导数，表示 x 变化时 f 变化的快慢

x=2 时，f'(2) = 4  // x 每增加 1，f 增加 4
```

**AI 中的应用**：梯度下降（训练模型用）

```
目标：找到让误差最小的参数

方法：
1. 计算当前参数的梯度（导数）
2. 沿着梯度的反方向更新参数
3. 重复直到收敛

类比：下山
- 你在山上，想走到最低点
- 每次看哪个方向最陡（梯度）
- 往那个方向走一步
- 重复直到到谷底
```

**面试考点**：
> **Q: 梯度下降是什么？**
> 
> **A**: 
> 训练模型的优化算法。
> 想象你在山上要走到最低点，梯度就是最陡的方向，
> 每次往梯度反方向走一步，重复直到最低点。
> 
> 在 AI 中，"最低点"就是误差最小的参数组合。

### 偏导数 (Partial Derivative)

**直观理解**：多变量函数，只让其中一个变量变化

```
f(x, y) = x² + y²
∂f/∂x = 2x  // 只对 x 求导，y 当常数
∂f/∂y = 2y  // 只对 y 求导，x 当常数
```

**AI 中的应用**：神经网络有多个参数，需要对每个参数求偏导

**前端需要知道吗**：知道概念就行，实际不用手算！

## 📊 3. 概率与统计（Probability & Statistics）

### 概率分布 (Probability Distribution)

**直观理解**：所有可能结果的概率

```javascript
// 抛硬币
P(正面) = 0.5
P(反面) = 0.5

// AI 应用：模型输出概率
模型预测：
P(这是猫) = 0.85
P(这是狗) = 0.15
```

### 期望值 (Expected Value)

**直观理解**：长期平均结果

```
抛骰子的期望值：
E = (1+2+3+4+5+6) / 6 = 3.5

AI 应用：强化学习中计算期望回报
```

### 贝叶斯定理 (Bayes' Theorem)

**公式**：P(A|B) = P(B|A) × P(A) / P(B)

**直观理解**：有了新证据后，更新概率

```
例子：疾病检测
- 已知：某病发病率 1%，检测准确率 99%
- 问题：检测阳性，真的得病的概率是多少？

直觉：99%？
实际：约 50%！（因为发病率太低）

贝叶斯定理就是用来算这个的
```

**面试考点**：
> **Q: 朴素贝叶斯分类器是什么？**
> 
> **A**: 
> 基于贝叶斯定理的分类算法。
> "朴素"指假设所有特征相互独立。
> 常用于垃圾邮件检测、文本分类。
> 
> 公式：P(类别 | 特征) = P(特征 | 类别) × P(类别) / P(特征)

### 常见分布

| 分布 | 用途 | 前端需要知道吗 |
|------|------|---------------|
| **正态分布** | 自然现象、误差分布 | ⭐ 知道名字 |
| **均匀分布** | 随机初始化 | ⭐ 知道概念 |
| **伯努利分布** | 二分类（是/否） | ⭐⭐ 理解 |

## 🔢 4. 信息论（Information Theory）

### 熵 (Entropy)

**直观理解**：不确定性的大小

```
例子：
- 抛硬币：熵高（结果不确定）
- 两面都是正面：熵低（结果确定）

公式：H = -Σ p(x) × log₂(p(x))
```

**AI 中的应用**：
- 决策树：选熵减最大的特征分裂
- 损失函数：交叉熵损失

### 交叉熵 (Cross Entropy)

**直观理解**：两个概率分布的差异

```
真实分布：[1, 0, 0]  // 肯定是猫
预测分布：[0.8, 0.1, 0.1]  // 80% 是猫

交叉熵越小，预测越准确
```

**面试考点**：
> **Q: 为什么分类问题用交叉熵损失？**
> 
> **A**: 
> 交叉熵衡量预测分布和真实分布的差异。
> 分类问题中，真实分布是 one-hot（正确类别为 1，其他为 0），
> 交叉熵越小，说明预测越接近真实标签。

## 🎯 面试数学题汇总

### 基础概念题（必会）

**1. 什么是向量？在 AI 中有什么用？**
```
向量是一组数字，表示事物的特征。
在 AI 中，文本、图片都可以转换成向量（Embedding），
方便计算相似度、做检索等。
```

**2. 余弦相似度的范围是多少？表示什么？**
```
范围：-1 到 1
1：完全相同
0：无关
-1：完全相反

AI 中常用于计算文本相似度。
```

**3. 梯度下降是什么？**
```
优化算法，用于训练模型。
想象下山，每次往最陡的方向走一步，
重复直到最低点（误差最小）。
```

**4. 什么是过拟合？怎么解决？**
```
过拟合：模型在训练数据上表现好，在新数据上表现差。
原因：模型太复杂，记住了训练数据而不是学习规律。

解决方法：
1. 增加训练数据
2. 简化模型
3. 正则化（L1/L2）
4. Dropout（深度学习）
5. 早停（Early Stopping）
```

### 进阶题（了解即可）

**5. 为什么神经网络用非线性激活函数？**
```
如果只有线性变换，多层网络等价于单层。
非线性激活函数（如 ReLU）让网络能学习复杂模式。
```

**6. 解释一下注意力机制（Attention）**
```
Attention 让模型在处理时关注重要的部分。
比如翻译"我喜欢吃苹果"，生成"apple"时，
模型会关注"苹果"这个词，而不是其他词。

公式：Attention(Q, K, V) = softmax(QK^T / √d) × V
（不用背公式，理解概念就行）
```

<details>
<summary>点击查看答案</summary>

我们要最小化损失函数，所以往反方向（下降最快）走。
