# AI 知识体系库 - 前端开发面试冲刺

> 📚 专为大厂前端开发打造的 3 个月 AI 学习路线图
> 
> 🎯 目标：从 AI 零基础到面试通关

---

## 📖 本书定位

**适合人群**：
- ✅ 有前端开发经验（3 年 +）
- ✅ 没系统学过 AI/机器学习
- ✅ 3 个月内要面试新工作
- ✅ 需要掌握 AI 相关面试考点

**不适合**：
- ❌ 想成为 AI 算法工程师（需要更深的数学和算法基础）
- ❌ 想从零开始学 Python（本书假设你会至少一门编程语言）
- ❌ 时间充裕想系统学习（本书是应试导向）

---

## 🎯 学习目标

3 个月后你应该能够：

1. **理解核心概念** - 能清晰解释 AI/ML/DL/LLM 的区别和联系
2. **掌握应用场景** - 知道前端能用 AI 做什么，怎么做
3. **应对面试问题** - 能回答 90% 的 AI 相关面试题
4. **实际项目经验** - 有 2-3 个可展示的 AI 项目

## 🔥 核心原则

### 学什么 vs 不学什么

| ✅ 必学 | ❌ 不学 |
|--------|--------|
| Prompt Engineering | 反向传播推导 |
| RAG 原理和实现 | 手推神经网络公式 |
| API 调用和集成 | 训练大模型 |
| AI 应用架构 | 复杂的数学证明 |
| 成本控制和优化 | 底层框架源码 |
| 面试高频考点 | 过时的 AI 技术 |

### 前端的优势

作为前端开发，你已经有这些优势：

1. **JavaScript/TypeScript** - 足够写 AI 应用
2. **HTTP/API 理解** - 调用大模型 API 很容易
3. **用户体验思维** - AI 应用也需要好的 UX
4. **工程化能力** - 部署、监控、优化都懂

### 前端的劣势（需要补）

1. **AI 术语不熟悉** - 需要快速扫盲
2. **缺少项目经验** - 需要快速做 2-3 个项目
3. **数学基础薄弱** - 只补面试够用的
